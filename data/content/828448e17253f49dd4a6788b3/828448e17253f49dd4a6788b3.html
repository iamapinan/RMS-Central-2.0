Samsung Galaxy Note 3 (Note III) ซัมซุง กาแลคซี่ โน้ต 3 เปิดตัว พร้อมเคาะราคาในไทยแล้ว 23,500 บาท พร้อมบทความ รีวิว samsung galaxy note 3


ได้ฤกษ์วางจำหน่ายในไทยอย่างเป็นทางการครับ สำหรับ Samsung Galaxy Note Note 3 (Note III) ที่เปิดตัวอย่างเป็นทางการ ในงาน Samsung Unpacked 2013 Episode 2 ที่เบอร์ลิน ประเทศเยอรมนี เมื่อวันที่ 4 กันยายน ที่ผ่านมา และในวันนี้ ก็เปิดตัวย่อมๆ ในไทยแล้วเช่นกัน ซึ่งทาง ซัมซุง ประเทศไทย ได้ประกาศราคา Samsung Galaxy Note 3 ออกมาแล้ว อยู่ที่ 23,500 บาท สำหรับความจุ 32 GB มีให้เลือก 2 สี ได้แก่ สีดำ Jet Black และ สีขาว Classic White

โดย Samsung Galaxy Note 3 (Note III) จะเปิด พรีออเดอร์ ในวันที่ 18-24 กันยายน 2556 และสามารถรับเครื่องได้ ในวันที่ 25-28 กันยายน 2556 หลังจากนั้น จะไปเปิดตัว และจำหน่ายในงาน TME 2013 Showcase วันที่ 3-6 ตุลาคม 2556 ครับ

สำหรับ Samsung Galaxy Note 3 รุ่นที่จำหน่ายในไทย จะเป็นรุ่นที่รองรับ 3G หรือรุ่นที่ใช้หน่วยประมวลผลแบบ Octa-Core Processor ความเร็ว 1.9 GHz (โมเดล N9000) นั่นเอง ส่วนรุ่นที่ใช้หน่วยประมวลผลแบบ Quad-Core Processor ความเร็ว 2.3 GHz ซึ่งเป็นรุ่นที่รองรับ LTE (โมเดล N9005) นั้น ยังไม่มีข้อมูลยืนยันครับว่า จะนำมาจำหน่ายในไทยด้วยหรือไม่

รีวิว Samsung Galaxy Note 3 (Note III)

ในวันเสาร์ที่ 14 กันยายน 2556 ที่ผ่านมา ทีมงานเว็บไซต์ Techmoblog ได้มีโอกาสได้ไปร่วมงาน ที่ทางซัมซุงได้จัดขึ้น ซึ่งงานดังกล่าวนั้น ได้เปิดโอกาสให้ทีมงานได้สัมผัสกับ สมาร์ทโฟนระดับเรือธงอย่าง Samsung Galaxy Note 3 เป็นกลุ่มแรก ดังนั้นจึงถือเป็นโอกาสดี ที่ทีมงานจะได้นำเอาสมาร์ทโฟนรุ่นดังกล่าว มารีวิวให้เพื่อนๆ ชาวเว็บไซต์ Techmoblog ได้ชมกัน เราลองมาดูกันครับว่า Samsung Galaxy Note 3 นั้น จะน่าสนใจ และ มีฟีเจอร์อะไรเด็ดๆ กันบ้าง

Samsung Galaxy Note 3 : วัสดุ และการออกแบบ

สำหรับ Samsung Galaxy Note 3 นั้น มาพร้อมกับหน้าจอขนาด 5.7 นิ้ว แบบ Super AMOLED Capacitive Touchscreen ความละเอียด 1920 x 1080 พิกเซล (Full HD) ซึ่งถือว่าเป็นหน้าจอที่มีความคมชัด และ สีสันสดใสเป็นอย่างมากเลยทีเดียว

โดยด้านบนของหน้าจอแสดงผล ประกอบด้วย ลำโพงสำหรับการสนทนา, Proximity Sensor, Ambient Light Sensor และ กล้องดิจิตอลด้านหน้า ความละเอียด 2 ล้านพิกเซล รองรับการใช้งานร่วมกับแอพพลิเคชั่นประเภท Video Call ได้เป็นอย่างดี 

ส่วนด้านล่างหน้าจอแสดงผล ประกอบด้วย ปุ่มควบคุมการทำงานมาตรฐาน 3 ปุ่มหลัก ได้แก่ ปุ่ม Home ซึ่งเป็นปุ่มแบบ Physical button กับ ปุ่ม Menu และปุ่ม Back ซึ่งเป็นปุ่มแบบ Capacitive button โดย 2 ปุ่มนี้สามารถใช้งานโดยใช้ปากกา S Pen สัมผัสได้แล้ว โดยรุ่นก่อนหน้าอย่าง Samsung Galaxy Note 2 ไม่สามารถทำได้

สำหรับด้านซ้ายของตัวเครื่อง จะประกอบด้วย ปุ่มสำหรับ เพิ่มลดเสียง รวมถึงหากสังเกตดีๆ ในบริเวณ ขอบด้านข้างของตัวเครื่อง จะมีดีไซน์เป็นเส้นๆ ซึ่งดีไซน์ดังกล่าว ได้รับแรงบันดาลใจมาจาก ด้านข้างของสันหนังสือ ที่มีกระดาษซ้อนกันนั่นเอง

ส่วนด้านขวาตัวเครื่อง ประกอบด้วย ปุ่ม Sleep / Wake หรือปุ่มสำหรับเปิดปิดเครื่อง โดยตัวเครื่อง Samsung Galaxy Note 3 นั้นจะมีความบางเพียง 8.3 มิลลิเมตรเท่านั้น

ด้านบนตัวเครื่อง ประกอบด้วย Infrared Port, ไมโครโฟนสำหรับ สนทนา และ ช่องสำหรับเชื่อมต่อชุดหูฟังขนาดมาตรฐาน 3.5 มิลลิเมตร

ส่วนด้านล่างตัวเครื่อง ประกอบด้วย ไมโครโฟนหลักสำหรับสนทนา และ ช่องสำหรับชาร์จไฟและเชื่อมต่อคอมพิวเตอร์ ที่รองรับ USB 3.0 รวมถึง ลำโพงภายนอก และ ช่องสำหรับเก็บปากกา S Pen

ด้านหลังตัวเครื่องนั้น มีการเปลี่ยนแปลงไปจากรุ่นก่อนหน้าพอสมควร โดยพื้นผิวด้านหลังมีการนำเอาหนังเทียม มาเป็นส่วนประกอบ ซึ่งแน่นอนว่า จะทำให้ดีไซน์ดูมีระดับมากขึ้นกว่าเดิมพอสมควรเลยทีเดียว

กล้องดิจิตอลด้านหลังของ Samsung Galaxy Note 3 นั้นมีความละเอียดอยู่ที่ 13 ล้านพิกเซล พร้อมไฟแฟลชแบบ LED ที่ช่วยเพิ่มประสิทธิภาพในการถ่ายภาพบริเวณที่มีแสงน้อย

เราลองมาถอดฝาหลัง กันเพื่อที่จะดูว่า ด้านในของตัวเครื่อง Samsung Galaxy Note 3 นั้น จะมีอะไรน่าสนใจกันบ้าง ซึ่งแน่นอนว่า เปิดมาก็จะพบกับ ด้านในดังภาพ

ในส่วนของด้านบน ก็จะเป็นช่องสำหรับใส่ซิมการ์ดแบบ microSIM card และ microSD Card ซึ่งรองรับสูงสุดได้ถึง 64 GB เลยทีเดียว ซึ่งรุ่นที่วางจำหน่ายในไทยนั้น จะมีความจุภายในตัวเครื่องอยู่แล้วที่ 32 GB

ด้านล่างจะเป็นแบตเตอรี่ ที่สามารถถอดเปลี่ยนได้ โดยแบตเตอรี่นั้นมีขนาดอยู่ที่ 3200 mAh ซึ่งจากการทดสอบของทีมงานพบว่า แบตเตอรี่ ค่อนข้างใช้งานได้ยาวนานพอสมควรเลยทีเดียวครับ

นอกจากนี้ ทีมงาน techmoblog ได้ลองทดสอบ ความแข็งแรงของ ฝาหลัง Samsung Galaxy Note 3 ดูแล้วก็พบว่า มีความแข็งแรงมากพอสมควร สามารถโค้งงอได้ โดยไม่มีรอยใดๆ ให้เห็นเลย

และสิ่งที่ขาดไม่ได้สำหรับ Samsung Galaxy Note 3 นั้น ก็คือ ปากก S Pen ซึ่งในรุ่นใหม่นี้ เราจะสามารถใช้งาน S Pen ได้อย่างหลากหลายมากยิ่งขึ้น ซึ่งจะมีอะไรเด่นๆ เกี่ยวกับ S Pen บ้าง เดี๋ยวลองชมในส่วนถัดไปครับ

แอพพลิเคชั่น และ การใช้งาน Samsung Galaxy Note 3

Samsung Galaxy Note 3 มาพร้อมกับ ระบบปฏิบัติการ Android 4.3 (Jelly Bean) และมี User Interface แบบ Touch Wiz Nature UX เวอร์ชั่น 2.0

โดยทาง Samsung ได้มีการติดตั้ง แอพพลิเคชั่นพื้นฐานที่จำเป็นสำหรับการใช้งาน มาให้อย่างครบถ้วน เรียกได้ว่า เมื่อเปิดเครื่องขึ้นมาแล้ว ก็สามารถเลือกใช้งานได้เลยทันที ไม่จำเป็นต้องดาวน์โหลดเพิ่มเติม

อีกหนึ่งจุดเด่นของระบบปฏิบัติการ Android นั่นก็คือ การปรับแต่งหน้าจอได้ตามใจชอบด้วย Widgets หลากหลายชนิด รวมถึงคุณสามารถดาวน์โหลด Widgets เพิ่มเติมได้ภายหลังด้วย

ภาพด้านบนนั้น เป็นอินเทอร์เฟส สำหรับการใช้งานด้านโทรศัพท์ รวมถึง หน้าจอสำหรับการจัดการรายชื่อผู้ติดต่อต่างๆ ของคุณ ซึ่งสามารถจัดกลุ่ม หรือค้นหา รายชื่อได้อย่างง่ายดาย

สำหรับเบราว์เซอร์บน Samsung Galaxy Note 3 นั้นจะมีทั้ง Android Browser และ Google Chrome แต่ เรายังคงสามารถดาวน์โหลด เบราว์เซอร์ อื่นๆ เพิ่มเติมภายหลังได้อีกมากมาย

จากภาพด้านบนเป็นหน้าจออของ แอพพลิเคชั่น นาฬิกา ซึ่งสามารถทำงานได้ทั้งการตั้งปลุก การดูเวลาทั่วโลก การจับเวลา หรือแม้แต่ การตั้งเวลาก็สามารถทำได้ด้วยเช่นเดียวกัน

แอพพลิเคชั่น ปฏิทินนั้นก็ถือว่า เป็นอีกหนึ่งแอพพลิเคชั่นที่ใช้งานกันได้บ่อยๆ โดยเฉพาะในเรื่องของ การเพิ่มตารางนัดหมาย รวมถึง ให้แจ้งเตือน เมื่อใกล้ถึงเวลาของเหตุการณ์สำคัญๆ ในแต่ละวันนั่นเอง

แอพพลิเคชั่นสำหรับบันทึกเสียงบน Samsung Galaxy Note 3 นั้นก็ถูกออกแบบมาให้ใช้งานได้อย่างง่ายดาย โดยเรายังคงสามารถปรับโหมดของการบันทึกเสียงได้ด้วย โดยจะมีทั้งการอัดเสียงปกติ การสนทนา หรือการ สัมภาษณ์ เป็นต้น และที่สำคัญ Samsung Galaxy Note 3 ยังมาพร้อมกับ ไมโครโฟนสองตัว ซึ่งเหมาะสำหรับการบันทึกเสียง สัมภาษณ์ เพื่อให้ได้เสียงที่ชัดเจน ทั้งในฝั่งผู้สัมภาษณ์ และ ผู้ถูกสัมภาษณ์ นั่นเอง

ส่วนแอพพลิเคชั่น เครื่องคิดเลข เมื่อใช้งานในแนวนอน จะมีสูตรคำนวณให้เลือกใช้ได้อย่างหลากหลาย และ บนหน้าจอขนาดใหญ่ของ Samsung Galaxy Note 3 ช่วยให้เราได้รับความรู้สึกเหมือนกำลังกด เครื่องคิดเลขจริงๆ เลยทีเดียว

และพิเศษสำหรับผู้ใช้ Samsung Galaxy Note 3 ทาง ซัมซุงเองได้นำเอาแอพพลิเคชั่น Sketchbook Pro มาให้ใช้งานกันได้แบบฟรีๆ ไม่จำเป็นต้องซื้ออีกต่อไป

สำหรับภาพด้านบนนั้น เป็นอินเทอร์เฟสของเครื่องเล่นเพลงบน Samsung Galaxy Note 3 โดยจะมีปุ่มควบคุมมาตรฐาน ไม่ว่าจะเป็นสุ่มเพลง ปรับเสียง ข้ามเพลง และ อื่นๆ 

ด้วยหน้าจอขนาดใหญ่บน Samsung Galaxy Note 3 ก็จะช่วยให้คุณสามารถรับชมคลิปวีดีโอ รวมถึง ไฟล์วีดีโอต่างๆ ได้อย่างเต็มตามากยิ่งขึ้น ซึ่งน่าจะถูกใจสำหรับผู้ที่ชื่นชอบการชมคลิปวีดีโอเป็นอย่างมาก

Samsung Galaxy Note 3 นั้นมาพร้อมกับระบบ GPS และ A-GPS ในตัว สามารถใช้งานร่วมกับแอพพลิเคชั่นประเภทการนำทาง ได้ทันที รวมถึงแอพพลิเคชั่น ในเครื่องอย่าง Google Maps ด้วยเช่นเดียวกัน

สำหรับ ฟีเจอร์​ Multi Window บน Samsung Galaxy Note 3 นั้นมีการปรับปรุงใหม่ ให้สามารถเปิดแอพพลิเคชั่น เดียวกันทั้งสองหน้าต่างได้ รวมถึง สามารก Copy ข้อมูล ข้ามหน้าต่างได้ด้วยเช่นเดียวกัน

แอพพลิเคชั่นจาก ซัมซุงอย่าง Story Album ก็ถือเป็นอีกหนึ่งแอพพลิเคชั่น ที่เหมาะสำหรับคนที่ชื่นชอบการถ่ายรูป เพราะหลังจากถ่ายเสร็จ คุณสามารถนำมาตกแต่งเป็นอัลบั้มภาพถ่าย ได้หลากหลายรูปแบบ ตามที่คุณต้องการ

แอพพลิเคชั่น S Health เป็นแอพพลิเคชั่น ที่จะเข้ามาช่วยดูแลสุขภาพของคุณ ซึ่งคุณสามารถตั้งเป้าหมาย ที่เกี่ยวข้องกับสุขภาพของคุณได้อย่างง่ายดาย โดยเฉพาะ การออกกำลังกาย

และสำหรับ แอพพลิเคชั่น Group Play บน Samsung Galaxy Note 3 นั้นได้มีการปรับปรุงให้ดียิ่งขึ้น โดยล่าสุดสามารถใช้งาน Group Play ในการแชร์ไฟล์วีดีโอ ได้แล้ว 

ซึ่งจากภาพด้านบนนั้น จะเป็นตัวอย่างการใช้งาน Group Play ในการแชร์วีดีโอ ซึ่ง เมื่อนำ Samsung Galaxy Note 3 มาวางต่อกัน ก็จะกลายเป็น หน้าจอขนาดใหญ่ในทันที

S Translator แอพพลิเคชั่น ที่จะเข้ามาช่วยลดความแตกต่างทางด้านภาษาของคุณ โดย เมื่อคุณพูด หรือ พิมพ์ ภาษาหนึ่งลงไป ระบบ จะทำการแปลไปเป็นอีกภาษาหนึ่งตามที่คุณต้องการ

S Voice แอพพลิเคชั่น สั่งงานด้วยเสียง ที่สามารถทำงานได้อย่างหลากหลาย ตั้งแต่การถามข้อมูลธรรมดาๆ เช่น สภาพอากาศ เวลา หรือสถานที่ ไปจนถึงการสั่งเพื่อควบคุมตัวเครื่องเช่น เปิดปิด Wi-Fi เปิดแอพพลิเคชั่น และ อื่นๆ อีกมากมาย 

S Beam อีกหนึ่งฟีเจอร์ ที่ทำให้การแชร์ไฟล์เป็นเรื่องง่าย เพียงแค่ นำตัวเครื่องด้านหลัง มาสัมผัสกัน ก็จะสามารถแชร์ภาพถ่าย วีดีโอ และ อื่นๆ อีกมากมายได้ในทันที

ซึ่งแน่นอนว่า แอพพลิเคชั่นที่กล่าวถึง นั้นเป็นเพียงส่วนหนึ่งเท่านั้น คุณยังคงสามารถดาวน์โหลดแอพพลิเคชั่น และ เกม อีกหลายแสนรายการผ่าน Google Play Store ได้ทุกที่ทุกเวลา

ฟีเจอร์ใหม่ ที่น่าสนใจของ S Pen บน Samsung Galaxy Note 3

Samsung Galaxy Note 3 นั้น มีการพัฒนาให้คุณสามารถใช้งาน S Pen ได้อย่างหลากหลายมากยิ่งขึ้น โดยจากภาพ จะเป็นฟีเจอร์ใหม่ที่เรียกว่า Air Command ซึ่งสามารถเปิดใช้ Air Command ได้จากการกดปุ่มบนปากกา S Pen นั่นเอง โดย Air Command นั้น จะมีฟีเจอร์ย่อยๆ อีก 5 รูปแบบ ลองมาดูครับว่ามีอะไรเด่นๆ บ้าง

Action Memo เป็นฟีเจอร์ที่ช่วยให้คุณเขียนคำสั่งต่างๆ ด้วยลายมือของตัวเอง เช่นหากคุณเขียนเบอร์โทรศัพท์ลงไป ตัวเครื่องจะทราบว่า นั่นคือหมายเลขโทรศัพท์ คุณสามารถสั่งงานเพื่อโทรออกในเบอร์ดังกล่าวได้ในทันที

Scrapbook หรือสมุดภาพส่วนตัว ซึ่งเป็นการ capture ภาพที่ชื่นชอบ (ไม่จำเป็นต้อง capture ทั้งหน้าจอ) แล้วนำมาเขียนเป็นข้อความส่วนตัว หรือบันทึกประจำวัน จากนั้น เก็บบันทึกไว้เป็นหมวดหมู่ เพื่อง่ายต่อการมาเปิดอ่านในภายหลัง

Screen Write หรือการขีดเขียนบนหน้าจอ จะเป็นเหมือนกับการ capture หน้าจอครับ แล้วสามารถเขียนข้อความ วาดรูป หรืออะไรก็ได้ ลงไปบนภาพๆ นั้น

และบางที แอพพลิเคชั่น, รูปภาพ, เอกสาร หรือไฟล์ต่างๆ ที่มีในตัวเครื่อง ก็มีมากมายจนเราหาไม่เจอ ซัมซุง จึงได้เพิ่มแอพพลิเคชั่น ที่มีชื่อว่า S Finder สำหรับค้นหาแอพพลิเคชั่นที่ต้องการ ในระยะเวลาสั้นๆ 

Pen Window หน้าต่างสำหรับ เรียกแอพพลิเคชั่นแบบ shortcut ไม่ว่าจะเป็น เครื่องคิดเลข, นาฬิกา, YouTube, โทรศัพท์, รายชื่อ, ChatON หรือเบราว์เซอร์ ซึ่งสามารถเปิดใช้งานขณะอยู่หน้าใดก็ได้

ทดสอบ แอพพลิเคชั่น S Note

จุดเด่นของ Samsung Galaxy Note 3 ก็คือ ปากกา S Pen ซึ่งมีแอพพลิเคชั่นที่รองรับการใช้งานโดยเฉพาะ นั่นก็คือ S Note ครับ โดย S Note บน Samsung Galaxy Note 3 นั้น ได้รับการปรับปรุงใหม่ มีฟังก์ชันให้ใช้งานหลากหลายมากขึ้น โดยมีเทมเพลตให้เลือกใช้งานมากมาย สะดวกและเหมาะสำหรับผู้ที่ชอบจด หรือขีดเขียนเป็นประจำ

นอกจากนี้ S Note ยังมีฟังก์ชัน ที่เหมาะกับพนักงานออฟฟิศมากขึ้น ด้วยการเพิ่ม Easy Chart สำหรับสร้างกราฟแบบง่ายๆ ทั้งกราฟแท่ง, กราฟเส้น หรือ Pie graph เรียกได้ว่า สะดวกมากทีเดียวครับ

ปากกา S Pen เพิ่มฟีเจอร์ใหม่ !

สำหรับท่านที่ใช้ Samsung Galaxy Note 2 อยู่ คงจะได้สัมผัสกับความสามารถของ ปากกา S Pen กันไปบ้างแล้ว ซึ่ง S Pen บน Samsung Galaxy Note 3 (Note III) นั้น ได้รับการปรับปรุงใหม่ ให้รองรับการใช้งานได้มากยิ่งขึ้น ซึ่งการกดที่ปุ่มบนปากกา S Pen จะเป็นการเปิดฟีเจอร์ ที่มีชื่อว่า Air Command ครับ โดยมีฟีเจอร์ย่อย รองรับทั้งหมด 5 แบบด้วยกัน ได้แก่

• Action Memo รายละเอียดทุกอย่างที่ถูกจดโดยใช้ปากกา S Pen จะถูกแปลงให้อยู่ใน format ที่ต้องการ ยกตัวอย่างเช่น ถ้าจดเบอร์โทรศัพท์ รายละเอียดเบอร์โทรจะถูกแอดเข้า contact นอกจากนี้ ยังสามารถค้นหาที่อยู่จากแผนที่, ค้นหาผ่านเว็บไซต์ และอื่นๆ อีกมากมาย

• Scrapbook เป็นศูนย์รวมคอนเทนต์ หรือข้อมูลต่างๆ จากเว็บไซต์, YouTube รวมไปถึง Gallery เพื่อให้ง่ายต่อการเปิดดูในภายหลัง ซึ่ง Scrapbook นี้ จะเก็บรายละเอียดทุกอย่าง รวมไปถึง URL ที่ผู้ใช้สามารถ refer กลับไปหาต้นฉบับได้

• Screen Writeเป็นการ capture หน้าเพจนั้น และสามารถเขียนรายละเอียดอื่นๆ เพิ่มเติมไปลงบนภาพที่ capture ได้

• S Finder ฟีเจอร์เพื่อการค้นหาข้อมูลในตัวเครื่อง โดยสามารถใส่เป็น keyword หรือใส่ฟิลเตอร์จำพวก วันที่, สถานที่ และอื่นๆ

• Pen Window ผู้ใช้สามารถใช้ปากกา S Pen ได้หลากหลาย นอกเหนือไปจากการขีดเขียน หรือจด แต่ยังสามารถใช้เปิดแอพพลิเคชั่น หรือลดขนาดหน้าต่างของแอพพลิเคชั่นได้ตามใจชอบ

Samsung KNOX และ Find My Mobile

Samsung KNOX โซลูชั่นเพิ่มความปลอดภัยทั้งในเรื่องธุรกิจ และความบันเทิง ซึ่ง Samsung KNOX จะเน้นเรื่องการรักษาข้อมูลที่เป็นความลับ แยกข้อมูลส่วนตัวออกจากธุรกิจ เพื่อเพิ่มความสะดวกในการใช้งานนั่นเอง

ส่วน Find My Mobile เป็นฟีเจอร์ค้นหาโทรศัพท์ และปิดการทำงานเมื่อทำหายหรือถูกขโมย โดยจะป้องกันตัวเครื่องไม่ให้ถูกรีเซ็ตแบบ Factory reset นอกจากนี้ ยังสามารถ track ตัวเครื่อง เพื่อค้นหาตำแหน่ง หรือลบข้อมูลที่จำเป็นออกจากตัวเครื่องได้

เคสแบบ official สำหรับ samsung galaxy note 3 ครับ โดยทำมาจากหนังเทียม มีสีสันให้เลือกมากมาย

S View Cover โดยเคสชิ้นนี้ จะมีหน้าจอขนาดใหญ่ สำหรับควบคุมการใช้งาน samsung galaxy note 3 แม้ตัวเครื่องยังถูกปิดด้วยตัวเคสอยู่ ซึ่งหน้าจอในส่วนนี้ สามารถใช้ปากกา S Pen ได้ด้วย มีสีสันให้เลือกมากมายเลยทีเดียว

Samsung Galaxy Note 3 (Note III) กับการวางจำหน่ายในไทย ?

ซัมซุง ประกาศวางจำหน่าย Samsung Galaxy Note 3 (Note III) ในวันที่ 25 กันยายนนี้ ใน 140 ประเทศทั่วโลก ซึ่งยังไม่มีรายละเอียดระบุว่า มีประเทศใดบ้าง ส่วนการวางจำหน่าย samsung galaxy note 3 ในไทย จะเปิด พรีออเดอร์ ในวันที่ 18-24 กันยายน 2556 และสามารถรับเครื่องได้ ในวันที่ 25-28 กันยายน 2556 หลังจากนั้น จะไปเปิดตัว และจำหน่ายในงาน TME 2013 Showcase วันที่ 3-6 ตุลาคม 2556 ครับ

ราคา Samsung Galaxy Note 3 (Note III) ในไทย ?

ในงานเปิดตัว samsung galaxy note 3 นั้น ยังไม่มีการเผยราคา samsung galaxy note 3 ออกมาอย่างเป็นทางการ แต่ล่าสุด ทาง ซัมซุง ประเทศไทย ได้เผยราคา Samsung Galaxy Note 3 ออกมาแล้ว อยู่ที่ 23,500 บาท ซึ่งแพงกว่า Samsung Galaxy Note 2 ตอนเปิดตัวเล็กน้อยครับ
