Google Documents

ความหมายของ Google Documents


Google Documents หรือที่เรียกอีกอย่างหนึ่งว่า Google Docs ซึ่งนับว่าเป็นการบริการของ Google ด้วยระบบออนไลน์ที่ใช้สามารถจัดการกับเอกสารในรูปแบบต่าง ๆ โดยไม่ต้องเสียเงินค่าลิขสิทธิ์ในการจัดซื้อโปรแกรม ทำให้เราสามารถจัดการเกี่ยวกับเอกสาร การพิมพ์รายงาน การทำสไลด์เพื่อพรีเซนต์ หรือจะจัดการเอกสารแบบ Spreadsheet ได้เหมือนกับ Excel  ไม่ต้องติดตั้งโปรแกรม Microsoft Office  สามารถสร้างงานเอกสารร่วมกันในแบบออนไลน์ เพียงแค่เรามีอีเมลของ Gmail และทำการเชื่อมต่ออินเทอร์เน็ตในระหว่างการใช้งาน ก็สามารถจัดการเอกสารได้อย่างมืออาชีพ

  Google Docs ทำงานเหมือนกับ Microsoft Office แต่งทุกย่างจะทำงานอยู่บ้านเว็บไซต์  สามารถทำงานได้ทันทีที่มีการเชื่อมต่ออินเทอร์เน็ต โดยไม่ต้องเสียเวลาติดตั้งโปรแกรมลงในเครื่อง เพียงแค่เข้าไปยัง Google Docs ก็สามารถสร้าง แก้ไข หรือเปิดอ่านเอกสารได้เลย โดยเอกสารที่จัดทำขึ้นจะถูกเก็บไว้ในเซิร์ฟเวอร์ของ Google ที่สำคัญคือสามารถแชร์เอกสารให้กับเพื่อน ๆ เพื่อแก้ไขข้อมูลไปพร้อม ๆ กัน


การเตรียมความพร้อมของเครื่อง

  ก่อนที่จะเริ่มทำงานใน Google Docs เราจะต้องเตรียมความพร้อมของเครื่องคอมพิวเตอร์ของเราเสียก่อน  โดยสิ่งที่จำเป็นบนเครื่องที่จะต้องมีเพื่อรองรับการใช้งาน Google Docs มีดังนี้

1.  ระบบปฏิบัติการ (Operation System-OS) สามารถใช้งานระบบปฏิบัติการ Windows, Mac OS X, GNU/Linux ฯลฯ

2.  โปรแกรมเป็นเว็บไซต์  (Web Browser) สามารถใช้งานร่วมกับ Web Browser เช่น Internet Explorer (IE), Mozilla Firefox, Safari หรือ Opera

3.  อินเทอร์เน็ต (Internet) ติดตั้งอินเทอร์เน็ตเพื่อเชื่อมต่อการใช้งานผ่านระบบออนไลน์ ซึ่งอินเทอร์เน็ตควรจะเป็นแบบ  ADSL หรืออินเทอร์เน็ตความเร็วสูง เพื่อช่วยให้การเชื่อมต่อไม่สะดุดและล่าช้าจนเกินไป


ความสามารถของ Google Docs

  Google Docs มีความสามารถมากมาย สามารถตอบสนองความต้องการในการใช้งานเอกสารได้อย่างสมบูรณ์ ดังรายละเอียดต่อไปนี้


1.  สร้างเอกสาร สเปรดชีต และงานนำเสนอแบบออนไลน์

  • สร้างเอกสารพื้นฐานแบบเริ่มต้นจากศูนย์ สามารถทำงานพื้นบานทุกประเภทได้อย่างง่ายดาย รวมถึงการทำรายการสัญลักษณ์แสดงหัวข้อย่อย การเรียงลำดับตามคอลัมน์ การเพิ่มตาราง รูปภาพ ข้อคิดเห็น สูตร การเปลี่ยนแปลงแบบอักษร และอื่นๆ โดยไม่มีค่าใช้จ่ายใดๆ
  • อัปโหลดไฟล์ที่มีอยู่แล้ว รองรับรูปแบบไฟล์ที่นิยมใช้กันส่วนใหญ่ ได้แก่ DOC, XLS, ODT, ODS, RTF, CSV และ PPT เป็นต้น
  • ใช้งานบนแถบเครื่องมือได้อย่างคุ้นเคย ทำให้การแก้ไขเป็นเรื่องง่าย ๆ เลือกตัวหนา ขีดเส้นใต้ เพิ่มสัญลักษณ์ในข้อย่อย เปลี่ยนแบบอักษร หรือรูปแบบตัวเลข เปลี่ยนสีพื้นหลังของเซลล์


2.  ใช้งานและทำงานร่วมกันในแบบเรียลไทม์

  • เลือกคนที่ท่านต้องการให้เข้าถึงเอกสารของท่านได้ เพียงป้อนที่อยู่อีเมลล์ของคนที่ท่านต้องการให้ใช้งานเอกสารที่ระบุร่วมกัน แล้วส่งคำเชิญไปให้เขาเหล่านั้น ก็สามารถใช้งานเอกสารร่วมกันได้
  • ใช้งานร่วมกันได้ทันที ทุกคนที่ท่านเชิญให้เข้ามาแก้ไขหรือดูเอกสาร สเปรดชีต หรืองานนำเสนอของท่าน สามารถเข้าถึงข้อมูลได้ทันทีที่เข้าสู่ระบบ
  • แก้ไขและนำเสนอร่วมกับบุคคลอื่นในแบบเรียลไทม์ สามารถดูและแก้ไขร่วมกันได้หลายคนในเวลาเดียวกัน มีหน้าต่างสนทนาบนหน้าจอสำหรับการแก้ไขเอกสารและสเปรดชีต เพื่อแสดงให้ท่านเห็นว่าใครแก้ไขอะไรและเมื่อใด และแล้วการดูงานนำเสนอพร้อมกันไม่ใช่เรื่องยากอีกต่อไป เนื่องจากใครก็ตามที่เข้ามาร่วมในงานนำเสนอต่างก็สามารถติดตามงานนำเสนอนั้นได้โดยอัตโนมัติ


3.  จัดเก็บและจัดระเบียบงานอย่างปลอดภัย

  • แก้ไขและเข้าถึงจากที่ไหนก็ได้ ไม่ต้องดาวน์โหลดสิ่งใด ท่านก็สามารถเข้าถึงเอกสาร สเปรดชีต และงานนำเสนอของท่าสนได้จากคอมพิวเตอร์เครื่องใดก็ได้ที่มีการเชื่อมต่ออินเทอร์เน็ต และมีบราวเซอร์มาตรฐานโดยไม่มีค่าใช้จ่ายใด ๆ
  • จัดเก็บงานของท่านได้อย่างปลอดภัย อุปกรณ์เก็บข้อมูลแบบออนไลน์ และการบันทึกอัตโนมัติ ช่วยให้ท่านไม่ต้องกังวลเรื่องฮาร์ดไดรว์เสียหรือไฟดับเพื่อข้อมูลจะถูกเก็บไว้ในเซิร์ฟเวอร์ขนาดใหญ่ของ  Google
  • บันทึกและส่งออกสำเนาได้อย่างง่ายดาย สามารถบันทึกเอกสาร และสเปรดชีต ของท่านไปยังเครื่องคอมพิวเตอร์ของท่านเองในรูปแบบ DOC, XLS, CSV, ODS, ODT, PDF, RTF และ HTML ได้
  • จัดระเบียบเอกสารของท่าน ค้าหาเอกสารของท่านได้อย่างง่ายดายด้วยการจัดระเบียบเอกสารในโฟลเดอร์ต่างๆ และสามารถลากและวางเอกสารต่างๆ ของท่านลงในหลายโฟลเดอร์ได้ตามที่ท่านต้องการ


4.  ควบคุมว่าใครสามารถดูเอกสารของท่านได้

  • เผยแพร่งานของท่านเป็นหน้าเว็บ ท่านสามารถเผยแพร่เอกสารของท่านแบบออนไลน์ได้โดยการคลิกเพียงครั้งเดียว ก็สามารถเผยแพร่ผลงานชิ้นสำคัญของท่านให้เป็นหน้าเว็บได้อย่างง่ายดาย
  • ควบคุมว่าจะให้ใครเห็นหน้าเว็บของท่านได้บ่าง สามารถเผยแพร่ข้อมูลได้ทั่วโลก หรือจำกัดเอกสารให้เห็นได้ในกลุ่มเพียงแค่สองสามคน หรือจะสั่งไม่ให้ใครเห็นเอกสารนั้นเลยก็ได้ ซึ่งก็แล้วแต่ท่านจะกำหนด นอกจากนี้ยังสามารถหยุดการเผยแพร่ข้อมูลได้ตลอดเวลา
  • โพสต์เอกสารขั้นบล็อกของท่าน เมื่อท่านสร้างเอกสารเสร็จ ท่าสามารถโพสต์เอกสารลงบล็อกของท่านได้ทันที
  • เผยแพร่ภายในบริษัทหรือกลุ่มของท่าน เมื่อใช้ Google Apps จะช่วยให้ใช้งานเอกสาร สเปรดชีต และงานนำเสนอที่สำคัญร่วมกันภายในบริษัทหรือกลุ่มของท่านได้ง่ายขึ้น
